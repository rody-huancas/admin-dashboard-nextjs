## DASHBOARD 

Dashboard template


### Stack
- NextJS
- TypeScript
- TailwindCSS
- ShadCN
- Lucide React
- React Hook Form
- Recharts
- Zod


### Installation
1. Clone repository
   ```
   git clone https://github.com/rody-huancas/admin-dashboard-nextjs
   ```
   

2. Install Dependencies
   ```
   pnpm install
   ```
   
3. Execute
   ```
   pnpm dev
   ```
